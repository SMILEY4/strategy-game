- directory "tools"
    stores small independent services/logic to work on game state
    examples
    - economy update
    - reveal area
    - create entity
    - create road
    - ...

- directory "event-system"
    proper/base event system, classes can subscribe on events and trigger events
    

- directory "core"
    small classes built on event system that use "tools" to implement game features
    examples:
    - resolve create city command
    - ...
